This is a task wdone according to the requirements for the application of software developer roile at HMCTS

This is the Frontend project written with Next JS (A ReactJS Framework)

To start this project, all that is needed is to run "npm run dev"

But, to be able to perform HTTP requests in this project, the local server of HMCTS backend task has to be running 
before HTTP requests can be made

This project also made use of redux as it's global state container.
